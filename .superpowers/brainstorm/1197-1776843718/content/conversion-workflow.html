HTML → Vue Conversion Workflow

Section 3 of 4 — Step-by-step untuk setiap template baru

1
Prep HTML

Bersihkan file HTML dari sumber manapun:

  • Hapus CDN script (analytics, tracking, jQuery)
  • Hapus competitor-specific JS/classes
  • Sisakan: struktur HTML + CSS visual saja
  • Catat: warna utama, font, section yang ada
2
Scaffold template
npm run make:template jasmine
→ buat JasmineTemplate.vue dari boilerplate
→ auto-register di registry.js
→ buka file, siap edit
3
Map sections HTML → Vue
HTML section
Vue / data
Hero / cover
groomName, brideName, coverPhotoUrl
Opening text
sectionEnabled('opening'), sectionData('opening')
Akad / Resepsi
events[] → event_date_formatted, location
Galeri foto
galleries[], galleryLayout (default)
RSVP form
SectionRsvp component (reuse)
Ucapan / guestbook
SectionMessages component (reuse)
4
Replace hardcoded values
#8B6914 (warna)
primary (dari composable)
"Cinzel Decorative" (font)
fontTitle (dari composable)
"Rizky & Sinta" (nama)
groomName & brideName
"Sabtu, 12 Juli 2025"
events[0].event_date_formatted
5
Setup DB record + test
  • Insert ke tabel templates: slug, name, tier=premium, default_config, demo_data, thumbnail_url
  • Test di TemplatePreviewModal dengan demo data
  • Cek semua section tampil benar
  • Cek kustomisasi warna/font bekerja
Catatan untuk HTML scraped dari kompetitor:
Fokus ambil CSS visual saja (animasi, layout, ornamen). Logic interaksi dibuang — diganti oleh composable. Hindari copy font/aset yang mungkin berlisensi.
